# Agentic Resume Stress-Tester

An AI-driven system designed to analyze resumes, identify technical vulnerabilities, and generate grounded interview questions using a multi-agent orchestrated pipeline.

## Project Structure
- `agents/`: LangGraph agent definitions (Analyst, Strategist, etc.)
- `api/`: FastAPI endpoints
- `ml/`: Synthetic data generation and model training
- `nlp/`: Tokenization, embeddings, and clarity scoring
- `parsers/`: Document extraction and section parsing
- `models/`: Pydantic data schemas
- `utils/`: Logging, configuration, and shared utilities
